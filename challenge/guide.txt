Yes — you are assigning two variables at once. This is called unpacking.

 Step-by-step with your example

File line:

nisha,19
1. Read line
line = "nisha,19"
2. Split it
line.split(",")

 Output:

["nisha", "19"]

This is a list with 2 items.

3. Unpacking happens
name, marks = ["nisha", "19"]

 Python does:

name = "nisha"
marks = "19"
 So what are name and marks?

They are just variables that store:

name → "nisha" (string)
marks → "19" (string → later converted to int
